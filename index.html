Инструкция по API

Документация.

Документация доступна по следующим ссылкам:

Swagger UI: /docs

ReDoc: /redoc

Для регистрации и аутентификации необходимо отправить POST запросы.

Ссылка для регистрации пользователя: /auth/register/

Ссылка для аутентификации пользователя: /auth/login/


Для получения списка валют и курса валют необходимо отправить GET запросы.

Ссылка для получения списка доступных валют: /currency/list/

Пример ссылки для расчета обменных операций: /currency/exchange/?from=USD&to=EUR&amount=1
